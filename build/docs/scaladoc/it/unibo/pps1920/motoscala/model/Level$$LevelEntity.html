    Contains all possible entities in one level, whit their characteristic, contains even the level map size and level index.

    All entity contain basic information needed for construct ecs respectively entity. The entity are serialized and deserialized, for reconstruct one specific level, the size are used for the rendering the map and the index is used for order level.
